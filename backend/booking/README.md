# BTL-NMCSDLL
Đề tài: Quản lý đặt phòng khách sạn (Booking)

## Backend 
Environment: python 3.7

## Installation
Download Mongodb 

### Install all packages for app back end

pip install -r requirements.txt

## Initial database
run migrate/init_db.py to insert default values

## Run app
run main.py
